# carbon-footprint
基于Vue2 + Router + Element UI 前后端分离的碳足迹计算器大作业

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
